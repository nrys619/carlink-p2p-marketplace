# Backend Roadmap

## Phase 1: Hosted MVP

- Auth: Supabase Auth or Firebase Auth
- DB: vehicles, favorites, messages, deals
- Storage: vehicle photos and vehicle document images
- Persistence: replace localStorage with hosted API calls

## Phase 2: Trust Layer

- eKYC status on profiles
- Seller ownership check from vehicle document upload
- Buyer payment intent status
- Deal timeline events
- Admin dashboard for dispute handling

## Phase 3: Vehicle Automation

- OCR API for paper vehicle inspection certificates
- Electronic vehicle inspection certificate reading flow
- Photo analysis API for equipment candidates
- Market price recommendation API

## Phase 4: Transaction Operations

- Payment/escrow provider integration
- Administrative scrivener assignment
- Title transfer document checklist
- Handover confirmation
- Release-of-funds trigger
